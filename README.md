# Blue Circus — Website (GitHub Pages)

This is a simple static site for the band **Blue Circus**.

## Quick start (GitHub Pages)

1. Create a repo (e.g. `blue-circus-site`).
2. Copy these files into the repo root.
3. In GitHub: **Settings → Pages**
   - Source: **Deploy from a branch**
   - Branch: **main** / **root**
4. Your site will appear at `https://<username>.github.io/<repo>/`.

## Customize

Edit `index.html`:
- Update the bio, social links, and contact email.
- Replace the Spotify / YouTube embed links.
- Add show dates in the “Shows” section.

Branding:
- Replace `assets/img/og.jpg` (social preview image)
- Replace `assets/img/hero.jpg` (background image)

## Optional

- If you use a custom domain, add a `CNAME` file containing your domain.
